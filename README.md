# Aplikasi Absensi & Pengajuan Izin Karyawan
King Royal Hotel Brebes

## Deskripsi
Aplikasi berbasis Android yang digunakan untuk absensi dan pengajuan izin karyawan.

## Teknologi
- Flutter
- Express.js
- MySQL

## Fitur
- Login
- Absensi Masuk & Pulang
- Pengajuan Izin
- Riwayat Absensi
